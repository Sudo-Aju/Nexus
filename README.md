# NEXUS

NEXUS: A Terminal-Based Life Operating System.

## Installation

1. Ensure you have Python 3.10 or higher installed.
2. Clone the repository.
3. Install the package and its dependencies:

   ```bash
   pip install .
   ```

## Database Initialization

Before running the application, you must initialize the database:

```bash
python init_db.py
```

## Usage

To start NEXUS, run the following command in your terminal:

```bash
nex
```

## Features

- **Productivity Dashboard**: Manage your tasks and projects in a terminal interface.
- **Task Management**: Create, edit, and track tasks.
- **Second Brain**: Store notes and ideas.
- **Secure Storage**: Keep secrets and sensitive information safe.
- **Terminal UI**: Fast and efficient keyboard-driven interface.
